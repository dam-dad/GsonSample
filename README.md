# GsonSample

Ejemplo de uso de la librería [Gson](https://github.com/google/gson) para [marshalling](https://en.wikipedia.org/wiki/Marshalling_(computer_science))/[unmarshalling](https://en.wikipedia.org/wiki/Unmarshalling) de objetos Java en JSON.

> Incluye ejemplo de un adaptador para la serialización/deserialización de instancias de la clase [Instant](https://docs.oracle.com/javase/8/docs/api/java/time/Instant.html).

1. Compilar el proyecto:

```bash
mvn compile
```

2. Ejecutar la aplicación:

```bash
mvn exec:java
```
